기타
- robot_state_publisher (launched by bringup.launch.py) publishes robot_state via parameter `robot_description` (package: robot_state_publisher) (launch file: transbot_bringup/launch/bringup.launch.py)
- TF: `transbot_base::base_node` 브로드캐스터가 odom -> base_footprint (child_frame_id) TF를 보냄; 런치에서 base_node가 실행되도록 등록됨.

비고 및 권장작업
- 일부 패키지(transbot_laser 등)는 노드/토픽이 프로젝트에 존재하나 현재 표는 직접 검색된 파일에서 확인된 항목만 포함합니다. 더 완전한 수집을 원하시면 `transbot_ws_ros2/src/**` 전체를 다시 grep으로 스캔해 누락된 노드를 추가로 수집하겠습니다.
- 파일 경로와 토픽 타입은 소스에서 직접 확인한 내용을 기준으로 합니다. 런타임에서 remapping 또는 launch 인수로 변경될 수 있으므로 실제 실행 환경에서 `ros2 node list` / `ros2 topic list`로 검증하는 것을 권장합니다.
